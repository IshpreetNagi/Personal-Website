B.O.O.T.A.Y

Visit Devpost Page

Details

This project was intended for the ‘Aye Aye Hackers’ hackathon, another hackathon hosted by Major League Hacking (MLH). The main goal for this hackathon was less on creating a solution to an actual problem, but rather on trying to create a project for which we could create an actual working prototype. To learn more about the project, please visit the Devpost page associated with it using the link provided above.

In the end, ‘B.O.O.T.A.Y: Box of Overly-protected Treasure And Yams’, is a security chest with two-step authentication. Fundamentally, the box was like any other safe as it required a password to be entered for it open. However, to add an extra security feature, it was made so that the box would require one to enter a peg leg along with entering the entry code. The peg leg would act as a kind of key that once paired with the correct pin number, would allow the box to open, thus revealing the treasure inside.

A working prototype of the actual security chest was successfully created. The chest worked exactly as stated, as it only opened once the entry pin had been entered along with the security key. The prototype was created using a box, an Arduino Uno, breadboard, a series of buttons, jumper cables, and thick wooden sticks. The pin was entered through a series of push buttons on a breadboard, with the key being entered into the box itself.
